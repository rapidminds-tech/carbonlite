How we calculate — Methodology

Carbon estimates use the Sustainable Web Design Model (SWDM v4).

Energy model: Data center (0.055 kWh/GB) + Network (0.059) + Device (0.080) = 0.194 kWh/GB.

Grid intensity: 480 gCO2e/kWh global average (Ember 2023 Global Electricity Review). Regional overrides available in settings.

Embodied carbon: +16% for hardware manufacturing (per SWDM v4).
